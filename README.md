# Random Password Generator

**Автор:** [Аделя Рашитова]

## Описание

GUI-приложение для генерации безопасных случайных паролей с возможностью настройки параметров и сохранением истории.

### Основные возможности:
- Генерация паролей с настраиваемыми параметрами
- Выбор типов символов (буквы, цифры, специальные символы)
- Регулировка длины пароля (4-32 символа)
- Сохранение истории в JSON
- Экспорт истории
- Копирование пароля в буфер обмена
- Гарантированное включение всех выбранных типов символов

## Требования

- Python 3.6+
- tkinter (входит в стандартную библиотеку Python)
